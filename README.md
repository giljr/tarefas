# Tarefas

Um projeto simples para demonstrar testes com **Cucumber** e **RSpec**.

## Clonar o projeto

```bash
git clone git@github.com:giljr/tarefas.git
cd tarefas
```
Rodar os testes

RSpec
```
bundle exec rspec
```
E o Cucumber:
```
bundle exec cucumber
```
